# 🧩 프로젝트 PRD --- REALGUARD Dataset App

**부제:** "딥페이크 시대, 진짜 얼굴을 지키는 데이터 플랫폼"

------------------------------------------------------------------------

## 1️⃣ 프로젝트 개요

### 📌 프로젝트 명

**REALGUARD --- Responsible Deepfake Defense Dataset**

### 🎯 목표

사용자가 본인 얼굴·음성 데이터를 직접 촬영하고 라벨링하여\
딥페이크 탐지 AI가 현실적인 데이터를 학습할 수 있도록 돕는\
**공익형 데이터 수집·분석 플랫폼** 개발

------------------------------------------------------------------------

## 2️⃣ 핵심 문제 정의

-   현재 딥페이크 탐지 모델은 많지만,\
    현실을 반영한 **윤리적이고 다양한 얼굴·음성 데이터**가 부족함.\
-   대부분의 공개 데이터셋은 **연예인, 스튜디오 조명, 정제된 환경**에
    한정되어 있음.\
-   따라서 실제 SNS·영상통화·일상환경에서 탐지 성능이 급격히 떨어짐.

👉 **REALGUARD는 "현실형·윤리형 데이터셋"으로 이 문제를 해결한다.**

------------------------------------------------------------------------

## 3️⃣ 주요 기능 요약

  -----------------------------------------------------------------------
  구분                    기능                    설명
  ----------------------- ----------------------- -----------------------
  **1. 데이터 생성**      얼굴·음성 촬영          앱 내 카메라·마이크로
                                                  사용자 데이터 직접 생성

  **2. 자동 라벨링**      환경/조명/표정 등       촬영
                          메타데이터 자동 수집    환경·조도·기기정보를
                                                  자동 라벨링

  **3. 개인정보 보호**    익명화·암호화 저장      사용자 ID와 데이터 분리
                                                  저장, AES 암호화

  **4. 데이터 업로드**    클라우드 저장 및 검수   Supabase/Firebase
                                                  Storage에 안전 저장

  **5. 탐지 AI 연동**     탐지 성능 개선          수집된 데이터로 모델
                                                  fine-tuning 가능

  **6. 리워드 시스템**    포인트/뱃지 제공        데이터 기여도에 따른
                                                  리워드 제공

  **7. 시각화 피드백**    기여도 및 탐지 향상     "당신 데이터로 모델
                          수치 표시               성능 +0.6% 향상" 식
                                                  피드백

  **8. 데이터셋 다운로드  필터 기반 검색·다운로드 인증된 연구자에게 공개
  (연구자용)**                                    API 제공
  -----------------------------------------------------------------------

## 4️⃣ 사용자 플로우

1️⃣ **회원가입 & 동의**\
\> 개인정보 보호 및 데이터 활용 동의 절차 → 앱 사용 시작

2️⃣ **데이터 촬영**\
\> 얼굴 영상 (정면, 측면, 다양한 조명) 및 음성 녹음

3️⃣ **자동 라벨링 & 업로드**\
\> OpenCV / Mediapipe 기반으로 표정, 조도, 배경 자동 인식

4️⃣ **익명화 처리 후 서버 업로드**\
\> 이름·계정 정보 분리, 익명 ID로 변환 저장

5️⃣ **리워드 및 분석 피드백**\
\> 업로드 후 리워드 지급 → 모델 향상 통계 피드백

6️⃣ **데이터셋 공개 (선택적)**\
\> 승인된 데이터만 연구자에게 제공

------------------------------------------------------------------------

## 5️⃣ 기술 스택 제안

  -----------------------------------------------------------------------
  계층                    기술                    설명
  ----------------------- ----------------------- -----------------------
  **Frontend (App)**      React Native (bare),    카메라·마이크 접근 및
                          Expo Camera 제외        미션형 UI

  **Backend**             FastAPI + Python        업로드 API, 데이터
                                                  암호화, AI 처리

  **Storage**             Supabase / Firebase     얼굴·음성 데이터 안전
                          Storage                 저장

  **Database**            PostgreSQL / Supabase   사용자 메타데이터,
                          DB                      리워드, 로그 관리

  **AI Engine**           PyTorch + FaceNet +     얼굴·음성 임베딩 추출
                          Whisper                 및 탐지모델 학습

  **Security**            AES256 + 익명화 처리    얼굴·음성 데이터 보호

  **Analytics**           Pandas + Plotly         사용자 데이터 통계 및
                                                  모델 향상 분석

  **Infra**               AWS EC2 / Render        서버 배포 및 API 관리
  -----------------------------------------------------------------------
