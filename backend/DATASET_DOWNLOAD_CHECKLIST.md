#### ✅ 확인해야 할 것:
- [ ] **연구 목적 vs 상업적 사용**: 대부분의 데이터셋은 연구 목적으로만 사용 가능
- [ ] **재배포 금지**: 데이터셋을 그대로 공유하면 안 됨
- [ ] **인용 필수**: 논문이나 README에 데이터셋 출처 명시 필요

#### 📋 주요 데이터셋 라이선스:
- **DFDC (Kaggle)**: 연구 목적, 재배포 금지
- **FaceForensics++**: 연구 목적, 논문 인용 필수
- **Celeb-DF**: 연구 목적
- **AI허브**: 사용 목적에 따라 승인 필요

### 2. 디스크 공간 확인

#### ✅ 확인해야 할 것:
- [ ] **다운로드 크기**: 원본 데이터셋 크기
- [ ] **압축 해제 후 크기**: 압축 해제하면 더 커질 수 있음
- [ ] **프레임 추출 후 크기**: 비디오 → 이미지 변환 시 크기 증가
- [ ] **여유 공간**: 최소 2배 이상 여유 공간 필요

#### 💾 예상 공간:
```
원본 비디오: 5GB
압축 해제: 5GB
프레임 추출: 10-20GB (비디오에 따라 다름)
최종 이미지: 1-2GB (샘플링 후)
─────────────────────────
필요 공간: 약 20-30GB (전체 프로세스)
```

### 3. 다운로드 시간 및 방법

#### ✅ 확인해야 할 것:
- [ ] **인터넷 속도**: 대용량 데이터셋은 시간이 오래 걸림
- [ ] **다운로드 방법**: 
  - Kaggle API (권장)
  - 직접 다운로드
  - BitTorrent (일부 데이터셋)
- [ ] **인증 필요**: Kaggle 계정, API 키 등
- [ ] **다운로드 제한**: 일일 다운로드 제한 확인

#### ⏱️ 예상 시간:
- 5GB 데이터셋: 100Mbps 기준 약 7분
- 50GB 데이터셋: 100Mbps 기준 약 1시간
- 느린 연결: 훨씬 오래 걸릴 수 있음

### 4. 데이터 품질 및 형식

#### ✅ 확인해야 할 것:
- [ ] **비디오 형식**: MP4, AVI 등 (코덱 호환성)
- [ ] **해상도**: 720p, 1080p 등 (처리 시간 영향)
- [ ] **프레임 레이트**: 30fps, 60fps 등
- [ ] **데이터 불균형**: REAL vs FAKE 비율 확인
- [ ] **메타데이터**: 레이블 파일, JSON 등 포함 여부

#### 📊 데이터셋 구조 확인:
```
dataset/
├── metadata.json      # 레이블 정보
├── real_videos/       # 실제 비디오
└── fake_videos/       # 딥페이크 비디오
```

#### ✅ 확인해야 할 것:

### 8. 다운로드 방법별 가이드

#### 방법 1: Kaggle API (권장)

```bash
# 1. Kaggle API 설치
pip install kaggle

# 2. API 키 설정
# ~/.kaggle/kaggle.json에 API 키 저장

# 3. 데이터셋 다운로드
kaggle competitions download -c deepfake-detection-challenge

# 4. 압축 해제
unzip deepfake-detection-challenge.zip
```

**장점**: 
- 빠른 다운로드
- 재개 가능 (중단 시 이어서 다운로드)
- 자동화 가능

**단점**:
- API 키 필요
- 계정 필요

#### 방법 2: 직접 다운로드

**장점**:
- 간단함
- API 키 불필요

**단점**:
- 느림
- 중단 시 처음부터 다시
- 브라우저 필요

#### 방법 3: BitTorrent (일부 데이터셋)

**장점**:
- 빠름 (여러 소스에서)
- 재개 가능

**단점**:
- 모든 데이터셋 지원 안 함
- 시드 필요

### 9. 데이터 검증

#### ✅ 다운로드 후 확인:
- [ ] **파일 무결성**: MD5/SHA256 체크섬 확인
- [ ] **파일 개수**: 예상 파일 수와 일치하는지
- [ ] **파일 크기**: 손상된 파일 확인
- [ ] **샘플 확인**: 몇 개 파일 열어서 내용 확인

#### 🔍 검증 명령어:
```bash
# 파일 개수 확인
find dataset/ -name "*.mp4" | wc -l

# 파일 크기 확인
du -sh dataset/

# 샘플 비디오 재생
ffplay dataset/real_videos/sample.mp4
```
